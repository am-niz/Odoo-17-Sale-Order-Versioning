DIFFERENT QUOTATION VERSIONS

By Accurate IT Solutions

This module empowers users to efficiently create and manage multiple versions of Quotations, ensuring that only the selected version is delivered. Streamline your sales process by maintaining control over order versions and delivering with precision.

Description with screenshots

The following screenshot demonstrates how to set up different Quotation versions.

In this screenshot, the Version and Version Date fields are automatically populated.

By clicking the Actions icon in the top-left corner, a dropdown menu appears with the option to Create New Version.

After selecting the Create New Version option, a wizard popup appears, allowing you to set the new version and modify the order lines.

After clicking the Create Version button, the newly created version is set as the current quotation.

After clicking the Versions button at the top center, the previous version is displayed in the list.

You can view the details of the listed quotation version by opening it.

After clicking the Restore button, a confirmation popup appears.

After clicking the OK button, the current quotation is updated to the restored version. Additionally, the count on the Versions button is updated.

After clicking the Versions button, you can view the available Quotation Versions.

Return to the quotation and click the Confirm button.

After clicking Confirm, a Delivery is created.

After clicking the Delivery button, you can view the delivery details of the selected Quotation Version.

To set the Delivery as another Quotation Version, navigate to the quotation version and click the Restore button.

After clicking the Restore button, the Delivery count is updated.

After clicking the Delivery button, you will see that different version deliveries are listed. The newly restored quotation version is set to a Ready delivery status, while the previous version is set to Cancelled.

After clicking the Ready state delivery, you can view the details of that delivery version.
